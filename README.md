# Hyprland MSI RTX 4070 Installer

Instalador automatizado de Hyprland optimizado para **MSI Pulse 16 AI C1V** con **RTX 4070 Mobile**.

## Características

- ✅ Backup automático de KDE Plasma
- ✅ Drivers NVIDIA optimizados + 165Hz
- ✅ Panel Waybar con monitor GPU (VRAM/Temp/Power)
- 🔌 **Detección de cargador AC/Batería**
- 🧠 Ollama local con CUDA (llama3.1:8b)
- 🎨 Tema oscuro Dracula
